\title{Score Generation using Generalized Iterated Function Systems} \author{Michael
Gogins \\ \texttt{michael.gogins@gmail.com}} \maketitle
%\pagestyle{scrheadings}

\begin{abstract}
Extending our earlier work on iterated function systems (IFS) music, we present generalized forms of IFS for more easily generating musical scores or even musical sounds. An IFS is a set of contractive mappings of a complete metric space onto itself. Every IFS has a fractal attractor that is a fixed point. The Collage Theorem proves that any set can be approximated, to any desired degree of precision, by the attractor of some IFS. This means that literally any score can be generated by some IFS, although it is seldom obvious how to construct an IFS to obtain a desired score. Here, I present an algorithm in C++ that generalizes IFS in several ways to make them easier to use for composing music. First, the functions comprising the IFS are extended from the usual affine transformations or simple functions to be closures (captures, in C++ parlance) that can implement any function and that can refer to variables outside the scope of the function, including the structure of the IFS itself. Second, the attractors of IFS have typically been approximated by randomly sampling the set of functions for a fixed number of iterations, or by deterministically descending the Cayley tree of functions to a fixed depth of iteration, to compute a quantized approximation of the fractal attractor. Here, the attractor is computed by deterministically descending the Cayley tree of functions not to a fixed depth of iteration, but rather to a minimum distance from the fixed point. This causes the computed attractor to consist of a finite set of discrete points. That such extended IFS are still contractive preserves the power of the method.
\end{abstract}
